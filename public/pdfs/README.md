# Carpeta de PDFs

Esta carpeta contiene los archivos PDF que se visualizarán en la aplicación.

## Archivos necesarios

Para que la aplicación funcione correctamente, coloca aquí los siguientes PDFs:

- `reunion-vida-ministerio.pdf`
- `reunion-fin-semana.pdf`
- `programa-limpieza.pdf`
- `programa-ppoc.pdf`
- `programa-salidas.pdf`
- `programa-responsabilidades.pdf`
- `grupos-predicacion.pdf`

## Cómo añadir PDFs

1. Coloca tus archivos PDF en esta carpeta
2. Asegúrate de que los nombres coincidan con los configurados en `src/App.tsx`
3. Los PDFs estarán disponibles automáticamente en la aplicación

## Formato recomendado

- Tamaño máximo: 10 MB por PDF
- Orientación: Vertical (portrait)
- Resolución: 150-300 DPI para buena calidad

## Nota

En el Milestone 5, estos archivos se subirán a Supabase Storage para gestión en la nube.
